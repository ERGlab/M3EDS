IMU Intrinsics

Xianwei Lv

简介：

IMU的误差模型
加速度计和陀螺仪的误差可分为确定性误差和随机误差,确定性误差可以事先标定，包括bias,scale,misalignment等. 随机误差通常假定噪声服从高斯分布,包括Gaussian white noise,random walker等. Scale可以看成是实际数值和输入值之间的比值.misalignment多轴传感器制作的时候，由于制作工艺的问题，会使得xyz可能不垂直. “White Noise”和“Bias”，都有连续时间模型和连续时间模型两种形式，两种模式可以相互转换，这里强调一下有的标定工具输出的是连续型（imu_utils），有的是离散型（kalibr_allan）。 我这里用imu_utils 标定 imu，得到的加速度的噪声和加速度随机游走噪声以及角速度的噪声和角速度的随机游走噪声。

方法与流程：

Allan 方差法是 20 世纪 60 年代由美国国家标准局的 David Allan 提出的,它是一种基于时域的分析方法。简要的流程如下:
1. 保持传感器绝对静止获取数据。
2. 对数据进行分段,设定时间段的时长。
3. 将传感器数据按照时间段进行平均。
4. 计算方差,绘制艾伦曲线。。

1.安装依赖项

sudo apt-get install libdw-dev

2.安装imu驱动,能够录制imu topic。

3.下载imu_utils和code_utils

imu_utils下载地址为:https://github.com/gaowenliang/imu_utils
code_utils下载地址为: https://github.com/gaowenliang/code_utils

4.在编译之前我们先注意三个问题

(1)安装ceres库,code_imu依赖ceres。
(2)不要同时把imu_utils和code_utils一起放到src下进行编译。由于imu_utils 依赖 code_utils，所以先把code_utils放在工作空间的src下面，进行编译。

5..编译

建立一个文件夹catkin_ws,内部文件夹src;先将code_utils放置于src内

cd catkin_ws/src
git clone https://github.com/gaowenliang/code_utils.git
cd ..
catkin_make
再把imu_utils也放到工作空间的src下面，不要删除之前的文件，再编译：
cd catkin_ws/src
cd catkin_ws/src
git clone https://github.com/gaowenliang/imu_utils.git
cd ..
catkin_make
编译安装完成

6.标定

1、让IMU保持静止两个小时,录制IMU数据;

rosbag record 话题名 -O imu.bag
2、然后根据需求对src/imu_utils/launch文件进行修改:主要包括话题名字、时长之类的；
3、播放录制好的bag,一定要倍速播放。
rosbag play -r 200 imu-.bag (200倍速度发布)
4、run launch
roslaunch imu_utils xxx.launch

5、最后标定结果文件保存在src里找到对应的yaml文件
